# assets

Drop the README demo here as `demo.gif`.

Suggested recording (~10–15s, in a real AL workspace):

1. Type `ttable` in a `.al` file → `Tab` to expand the snippet.
2. Introduce an error → diagnostics underline appears within a second.
3. `Ctrl+.` → **AL: Build package** → status-bar spinner → success toast.

Capture tips:
- Record at a readable size (≈1280×720), keep it short, loopable.
- Tools: [`asciinema`](https://asciinema.org) + `agg` for terminal, or
  [Peek](https://github.com/phw/peek) / macOS screen-record → convert to GIF
  with `ffmpeg`/`gifski`. Keep the file under ~5 MB so it loads on the README.
